# attitude - uncomplicated time tracker

```bash
# Begin a timer on <project>.
$ tt start <project>

# Stop a timer on <project>.
$ tt stop <project>

# Cancel running timer or delete last entry on <project>.
$ tt start <project>

# Summarize time spent on <project>.
$ tt report <project>
```
